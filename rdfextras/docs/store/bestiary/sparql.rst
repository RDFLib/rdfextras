=========================================================
SPARQL :: a read-only Store for wrapping SPARQL endpoints
=========================================================


This is a read-only RDFLib Store around Ivan Herman et al.'s SPARQL service wrapper.

This version works with vanilla SPARQLWrapper installed by ``easy_install`` or similar.
